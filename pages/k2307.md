# Agent / Agente de propreté urbaine (K2307)

**Autres appellations :** Agent / Agente anti-graffiti, Agent / Agente de lutte contre les pollutions visuelles, Agent / Agente de nettoiement d'espace urbain, Agent / Agente de propreté, de nettoiement et de collecte, Agent / Agente de salubrité, Agent / Agente d'entretien des quartiers, Agent / Agente d'entretien d'espaces publics, Agent / Agente d'entretien du domaine public, Agent / Agente d'entretien et de nettoyage urbain, Agent / Agente d'hydrogommage

## Définition
L'Agent de propreté urbaine, un acteur essentiel pour maintenir nos villes propres et accueillantes. Assure le nettoyage et l'entretien des espaces publics urbains pour garantir un environnement propre et agréable Participe à la lutte contre les pollutions visuelles en nettoyant les graffitis et autres marquages indésirables Contribue à la salubrité publique en vidant régulièrement les poubelles et en gérant les points de collecte Peut conduire des engins légers de nettoiement pour accéder et nettoyer des zones difficiles

## Accès au métier
Cet emploi est accessible avec un niveau CAP dans le domaine de la propreté, voire sans diplôme.
