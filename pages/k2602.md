# Conseiller / Conseillère funéraire (K2602)

**Autres appellations :** Agent / Agente d'amphithéâtre, Agent / Agente de chambre funéraire, Agent / Agente de chambre mortuaire, Agent / Agente de formalités funéraires, Agent / Agente de funérarium, Agent / Agente technique funéraire, Assistant / Assistante funéraire, Assistant / Assistante funéraire de plateau d'assistance, Assistant commercial / Assistante commerciale funéraire, Chargé / Chargée des formalités funéraires

## Définition
Le conseiller funéraire est essentiel au service des familles en deuil. Accompagne les familles dans l'organisation des obsèques Conseille sur les différentes prestations funéraires disponibles Gère les formalités administratives liées au décès Assure la coordination entre les différents intervenants (cimetière, crématorium, etc.) Propose des solutions personnalisées adaptées aux besoins et aux budgets Peut participer à la vente de monuments et d'articles funéraires

## Accès au métier
Cet emploi est accessible avec la formation de conseiller funéraire et assimilé.
