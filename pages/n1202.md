# Agent / Agente de transit (N1202)

**Autres appellations :** Agent / Agente consignataire de navire, Agent / Agente de fret aérien, Agent / Agente de transit aérien, Agent / Agente de transit export, Agent / Agente de transit import, Agent / Agente de transit maritime, Consignataire maritime, Employé / Employée de consignation maritime, Employé / Employée de cotation maritime, Employé / Employée de courtage maritime

## Définition
L'agent de transit assure la gestion des opérations de transport international de marchandises (à l'import comme à l'export). Organise et supervise le transport de marchandises à l'international Coordonne toutes les étapes de la prestation de transport avec les différents acteurs du transport maritime, aérien et terrestre Assure le respect des réglementations douanières et de sûreté Gère les aléas qui peuvent survenir lors du transit Gère la documentation nécessaire pour le transit des marchandises Contribue à garantir une prestation qui respecte les objectifs en termes de qualité de service, de satisfaction client et de rentabilité.

## Accès au métier
Cet emploi est accessible à partir d'un diplôme de niveau BAC +2 en transport logistique ou commerce international. La pratique de l'anglais est exigée, celle d'une autre langue étrangère peut apporter un plus.
